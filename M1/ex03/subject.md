# Exercício

Escreva um programa que imprime o conteúdo de uma lista my_list, um item por
linha:

 - Exemplo do arquivo lists.py:

    - my_list = ['item1', 'item2', 'item3', 'item4']

    - Exemplo de saída esperada:

      ?> python3 lists.py

        - item1

        - item2

        - item3

        - item4

      ?>
